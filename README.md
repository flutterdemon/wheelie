## Installation

Install my-project with flutter

```bash
  flutter create my-project
  cd my-project
```
    
## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  flutter pub get
```

Start the server

```bash
  flutter run
```
